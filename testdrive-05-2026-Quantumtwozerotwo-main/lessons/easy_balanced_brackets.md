# Простое задание: Balanced Brackets

Нужно реализовать функцию:

```python
def is_balanced(text: str) -> bool:
```

Она должна возвращать `True`, если со скобками всё в порядке, и `False`, если нет.

Проверяем только скобки:
- `()`
- `[]`
- `{}`

Все остальные символы нужно игнорировать.

## Примеры

```python
is_balanced("") == True
is_balanced("()") == True
is_balanced("([])") == True
is_balanced("(]") == False
is_balanced("([)]") == False
is_balanced("text { [ ok ] }") == True
```

Работайте по шагам:

1. Пустая строка даёт `True`.
2. Одна пара круглых скобок даёт `True`.
3. Открывающая без закрывающей даёт `False`.
4. Закрывающая без открывающей даёт `False`.
5. Вложенные круглые скобки работают.
6. Добавьте квадратные скобки.
7. Добавьте фигурные скобки.
8. Проверьте неправильный порядок, например `([)]`.
9. Проверьте, что обычные символы игнорируются.

Порядок для пары:

1. Откройте `easy_balanced_brackets_task/`.
