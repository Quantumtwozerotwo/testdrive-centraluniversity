# Шаблон для работы в паре

Это готовая папка, с которой можно сразу начинать урок.

Что делать:
1. Зайдите в эту папку.
2. Откройте `test_solution.py`.
3. Раскомментируйте или допишите один тест.
4. Запустите `pytest -q`.
5. Допишите минимум кода в `solution.py`.
6. Когда тест стал зелёным, поменяйтесь ролями.

Команды:

```bash
cd lessons/template_pair_task
pytest -q
```

Если хотите, можете копировать эту папку под каждую новую задачу.
